IBM Data Science Professional Certificate Capstone Project

My capstone project for the IBM Data Science Professional Certificate focused on grouping natural neighbourhoods in Edinburgh based on the current population of food venues using data from the Foursquare Places API and from Edinburgh Council.

You can view the end product of this project by clicking on the map link below. This map shows the clusters my project sorted the natural neighbourhoods in Ednburgh into, based on the types of food venue that are present there.

Map

Project Notebook

During this project I learned to use folium, the Foursquare Places API and geodataframes to analyse geographical data. I sorted points into non-uniform shapes and thought about how to find all of the venues in each area. I plan to further the value of this project by refining the model using data about the price of the venues as well as popularity.
