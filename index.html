IBM Data Science Professional Certificate Capstone Project

My capstone project for the IBM Data Science Professional Certificate focused on grouping natural neighbourhoods in Edinburgh based on the current population of food venues using data from the Foursquare Places API and from Edinburgh Council. The project used k-means clustering using scikit-learn to cluster the neighbourhoods.

This map is the final product of the project and summarises the groups that natural neighbourhoods were sorted into.

During this project I learned to use folium, the Foursquare Places API and geodataframes to analyse geographical data. I sorted points into non-uniform shapes and thought about how to find all of the venues in each area. I plan to further the value of this project by refining the model using data about the price of the venues as well as popularity.
